DOM : Document Object Model

  • element : un objet logique représentant une balise affichable dans la page.
  • document. createElement nom
    Retourne un nouvel élément DOM vide
  • parent. appendChild fils
    Ajoute l'élément fils dans l'élément parent.
  • parent. insertBefore enfant ref
    Ajoute l'élément fils dans l'élément parent juste avant l'élément ref.
  • element. innerHTML
    Contient le HTML brut du contenu de element. Permet également de le modifier.
  • element. getAttribute nom
    Retourne la valeur de l'attribut nom de element.
  • element. setAttribute nom valeur
    Modifie la valeur de l'attribut nom de element.
  • element. classList.add classe
    Ajoute la classe donnée à element. (Modifie l'attribut class)
  • element. classList.remove classe
    Retire la classe de element.
  • element. classList.toggle classe
    Ajoute la classe donnée à element si elle n'y est pas ou la retire sinon.
  • element. classList.contains classe
    Indique si la classe est appliquée à element.
